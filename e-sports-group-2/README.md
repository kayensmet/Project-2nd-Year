# Controls and used libraries for Fish to Dish

In this file you’ll find all the controls for the game. These controls are all based on AZERTY keyboard inputs.
To run the game you will need to run <u> **npyfile.py**</u>.

**Controls:**

**Moving mouse in x-direction:** This movement will turn the player left and right.

**Left mouse button press:** This will attack a fish if it’s in reach of the player.

**Z:** This will move the player forwards.

**S:** This will move the player backwards.

**Q:** This button is for strafing left.

**D:** This button is for strafing right.

**E:** This will open the players inventory.

**M:** This will take the player to the maze where they need to find the proper ingredients.

**P:** This will take the player to the shops where they can sell their sushi if it’s in their inventory.

**Escape:** When pressed in the main game you’ll be taken back to the main menu, pressed again you’ll go back to the main game.

**Space:** After pressing space the game will close.

**Used Libraries:** numba, numpy, Pillow, PySDL2, pysdl2-dll, pyautogui
